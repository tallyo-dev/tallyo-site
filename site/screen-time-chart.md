## The rules so the chart doesn't spiral

<ol class="article-steps">
  <li><strong>Approval before credit.</strong> Chores only count after a parent checks them. Kids can't self-award screen time.</li>
  <li><strong>Daily cap applies regardless of coins saved.</strong> Even if they have 20 coins, the daily cap is the cap. Coins carry over to tomorrow or bigger rewards.</li>
  <li><strong>Screen time is a reward, not a right.</strong> Baseline expectations (homework, meals) must be met before any coins can be spent.</li>
</ol>

<div class="article-callout">
  <div class="panel-header">Make it stick</div>
  <p>Set one daily "approval moment" — right after school or before dinner. This is when chores get checked and coins are added. Consistency matters more than the exact point values.</p>
</div>

## What if they don't do the chores?

If chores aren't done, no coins are earned. No coins means no screen time beyond whatever baseline your family allows.

This is different from "taking away" screen time as punishment. The chart is neutral: you earn it or you don't. Most kids find this much easier to accept than a rule that can change based on mood.

If your child is skipping chores consistently, use the <a href="/how-to-make-kids-do-chores/">how to make kids do chores</a> guide to troubleshoot the system.

## FAQ

<ul class="faq-stack">
  <li class="faq-card">
    <h3>How do you make kids earn screen time?</h3>
    <p>Use chores + approval + a simple conversion (coins to minutes). Redeem in small blocks with daily caps based on age. The key is that the rules are set in advance and don't change based on the situation.</p>
  </li>
  <li class="faq-card">
    <h3>Is earned screen time better than time limits?</h3>
    <p>Many families use both together: earned screen time creates the path to "yes," and daily caps set the ceiling. Limits alone answer "how much" — earning also answers "how to get there."</p>
  </li>
  <li class="faq-card">
    <h3>How much screen time is appropriate by age?</h3>
    <p>There's no single right answer, but a common starting point is 20–30 minutes on school days for ages 6–8, scaling up to 45–60 minutes for ages 11–12. Weekends can be more flexible. The chart helps because kids earn within whatever cap you set.</p>
  </li>
  <li class="faq-card">
    <h3>What if my child saves up a lot of coins?</h3>
    <p>The daily cap still applies — saving coins means they can redeem bigger rewards on weekends or save for non-screen rewards (choosing the family movie, dessert, etc.). Having coins to save also keeps motivation high between payouts.</p>
  </li>
  <li class="faq-card">
    <h3>Should I use a paper chart or an app?</h3>
    <p>Paper charts are easy to start and visible on the fridge. App-based systems add automatic coin tracking, parent approvals, and streaks — which help maintain consistency when paper charts fall off the fridge and get forgotten.</p>
  </li>
</ul>
