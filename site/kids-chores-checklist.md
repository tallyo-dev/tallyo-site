## Daily chores checklist (simple)

- [ ] Room reset (5 minutes)
- [ ] Clear dishes / wipe table (3 minutes)
- [ ] Prep for tomorrow (2 minutes)

For a full routine, see <a href="/daily-chore-list-for-kids/">daily chore list for kids</a>.

## Weekly chores checklist

- [ ] Vacuum a room
- [ ] Change bed sheets
- [ ] Bathroom sink + mirror
- [ ] Fold laundry (simple)
- [ ] Full room reset

## Make the checklist work (3 rules)

1) Same time every day
2) Approval before credit
3) Earned rewards (capped)

Use this hub for rewards: <a href="/reward-system-for-kids/">reward system for kids</a>.
