## 1) Start smaller than you think

If a chore regularly turns into a fight, it’s usually too big or too vague.

Instead of:
- "Clean your room"

Try:
- "Clothes in hamper + toys in bin" (5 minutes)

A 5-minute win builds momentum.

## 2) Use a daily routine (anchor moment)

Chores work best when they happen at the same time every day.

Pick one anchor moment:
- right after school
- before dinner

Use this template: <a href="/daily-chore-list-for-kids/">daily chore list for kids</a>.

## 3) Make chores visible (a simple chart)

If kids can’t see what to do, parents end up repeating themselves.

Start with a simple <a href="/chore-chart-for-kids/">chore chart</a>. If you need ideas by age, use <a href="/chores-by-age/">chores by age</a>.

## 4) Add approvals (no approval, no credit)

Approvals prevent "half done" tasks and keep the system fair.

## 5) Add earned rewards (capped)

Rewards aren’t bribery when they’re predictable.

Start with:
- <a href="/earned-screen-time/">earned screen time</a> in small blocks
- a simple weekly payout (see <a href="/allowance-for-chores/">allowance for chores</a>)

Or use one hub page: <a href="/reward-system-for-kids/">reward system for kids</a>.
